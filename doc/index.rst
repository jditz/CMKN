Introduction
------------

This package implements the convolutional motif kernel network introduced by Jonas C. Ditz et al., "Convolutional Motif Kernel Network", 2021.


Requirements
------------

Python >= 3.7
PyTorch >= 1.8.1
NumPy >= 1.19.1
BioPython >= 1.78
